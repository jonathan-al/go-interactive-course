# Exercise 1: Hello World

## Objective

Write your first Go program that prints `Hello, World!` to the console.

## Instructions

Open the file `starter.go`. The `main` function currently prints `"Hello, Go!"`. Your task is to **change the string** so the program prints:

```
Hello, World!
```

## Expected Output

```
Hello, World!
```

## Hints

- Modify the argument inside `fmt.Println(...)` to `"Hello, World!"`.
- Go strings are enclosed in double quotes `"..."`.
- `fmt.Println` automatically adds a newline at the end.
