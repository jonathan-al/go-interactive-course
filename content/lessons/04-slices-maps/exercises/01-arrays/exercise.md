# Exercise 01: Arrays

## Objective

Learn how to declare an array, access its elements, and get its length.

## Instructions

1. Declare an array of 5 integers with values `10, 20, 30, 40, 50`.
2. Print the length of the array using `len()`.
3. Print each element of the array on a separate line.

## Expected Output

```
Length: 5
10
20
30
40
50
```
