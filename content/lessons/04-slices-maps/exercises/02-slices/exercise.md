# Exercise 02: Slices

## Objective

Learn how to create a slice, append elements, and use slicing to extract sub-slices.

## Instructions

1. Create a slice with initial values `1, 2, 3`.
2. Append `4` and `5` to the slice.
3. Print the resulting slice.
4. Create a sub-slice from index 1 to 3 (exclusive) and print it.

## Expected Output

```
Slice: [1 2 3 4 5]
Sub-slice: [2 3]
```
