# Exercise 01: Structs

## Objective

Learn how to define a struct type, create instances, and access fields.

## Instructions

1. Define a `Person` struct with two fields: `Name` (string) and `Age` (int).
2. Create a `Person` instance with `Name: "Alice"` and `Age: 30`.
3. Print the struct using the default format.
4. Print the fields individually in the format `Name: <name>, Age: <age>`.

## Expected Output

```
{Alice 30}
Name: Alice, Age: 30
```
